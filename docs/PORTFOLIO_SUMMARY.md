# Portfolio Summary

REST Radar is a custom WordPress QA/security review plugin that inspects registered REST API endpoints, highlights risky permission patterns, generates QA bug report drafts, and supports regression comparison through endpoint snapshots.

It also includes a non-destructive Endpoint Shield layer that can temporarily protect selected routes without modifying third-party plugin or theme files.
