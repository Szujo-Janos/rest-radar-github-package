# Portfolio Summary

REST Radar is a custom WordPress QA/security review plugin that inspects registered REST API endpoints, highlights risky permission patterns, generates QA bug report drafts, supports regression comparison through endpoint snapshots, and adds a human review / finding triage workflow.

It also includes a non-destructive Endpoint Shield layer that can temporarily protect selected routes without modifying third-party plugin or theme files.

## Key portfolio points

- Built a WordPress REST API endpoint scanner
- Added risk classification for permission callback and public write-access patterns
- Added source detection for core, plugin, theme, MU plugin, and unknown routes
- Added QA-ready CSV and Markdown export
- Added snapshot comparison for plugin/theme update regression testing
- Added Endpoint Shield for temporary non-destructive endpoint mitigation
- Added review statuses, reviewer notes, severity override, and retest-required logic
- Redesigned the dashboard into an operational triage overview
- Added privacy-aware IP anonymization option and safer snapshot storage limits
