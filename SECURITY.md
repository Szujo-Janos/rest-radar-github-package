# Security Policy

REST Radar is a review and mitigation tool for WordPress REST API endpoints.

## Reporting issues

If you find a security issue in REST Radar itself, please report it privately before public disclosure.

## Scope

REST Radar helps identify potentially risky REST API routes, missing permission callbacks, and unexpected endpoint changes. Findings should be manually verified before being treated as confirmed vulnerabilities.

## Important note

Endpoint Shield is a non-destructive mitigation layer. It does not modify third-party plugin or theme source files.
